## 🎛️ PI参数调优指南

### 当前默认参数
```c
Kp = 8.0
Ki = 0.5
integral_max = 2000.0
integral_min = -2000.0
```

### 调优步骤

#### Step 1：调整 Kp（比例系数）
**目的**：改善响应速度

1. 将 Ki 暂时设为 0
2. 从小到大调整 Kp：5.0 → 8.0 → 10.0 → 15.0
3. 观察速度响应曲线

**现象分析**：
- Kp 太小 → 响应慢，无法达到目标速度
- Kp 合适 → 快速响应，轻微超调
- Kp 太大 → 剧烈震荡，不稳定

**推荐值**：找到刚好不震荡的最大Kp值

#### Step 2：调整 Ki（积分系数）
**目的**：消除稳态误差

1. 保持上一步确定的 Kp 值
2. 从小到大调整 Ki：0.1 → 0.5 → 1.0 → 2.0
3. 观察稳态误差

**现象分析**：
- Ki 太小 → 稳态误差大，速度总是偏低
- Ki 合适 → 稳态误差小，速度稳定
- Ki 太大 → 积分过强，容易震荡

**推荐值**：使稳态误差 < 5% 的最小Ki值

#### Step 3：调整积分限幅
**目的**：防止积分饱和

1. 如果启动时PWM长时间饱和（= 4000）→ 减小 `integral_max`
2. 如果稳态误差较大 → 增大 `integral_max`

**推荐值**：`integral_max = 1000 ~ 3000`

### 调优示例

**场景1：轻载（空载运行）**
```c
Motor_Left.kp = 10.0f;
Motor_Left.ki = 0.8f;
Motor_Left.integral_max = 1500.0f;
```

**场景2：重载（载重或爬坡）**
```c
Motor_Left.kp = 12.0f;
Motor_Left.ki = 1.5f;
Motor_Left.integral_max = 2500.0f;
```

**场景3：高速运行（> 500 mm/s）**
```c
Motor_Left.kp = 8.0f;
Motor_Left.ki = 0.5f;
Motor_Left.integral_max = 2000.0f;
```
